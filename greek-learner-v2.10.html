How to install a better Greek voice

Windows: Go to Settings → Time & Language → Speech → Manage voices → Add voices → select Greek. The available quality is basic, but it will sound more natural than the fallback.

macOS: System Settings → Accessibility → Spoken Content → System Voice → Manage Voices → tick Greek (Greece), download "Melina (Enhanced)" or any voice marked Premium.

Android: Settings → System → Languages & input → Text-to-speech → Google → Install voice data → Greek.

iOS: Usually preinstalled. Settings → Accessibility → Spoken Content → Voices → Greek — download the Enhanced voice if available.

After installing, reload this page and pick the new voice from the dropdown above.

Ordering like a local

Greeks take their coffee very seriously — a freddo espresso on ice is the standard summer order. At a bakery (φούρνος), a τυρόπιτα or σπανακόπιτα is a beloved snack at any hour. Tap 🔊 on any item to hear it pronounced.

🌯

Souvlaki vs Gyros — not the same thing

Easy to confuse, but the difference is in the cooking. Souvlaki is cubed meat marinated and grilled on a skewer — eaten off the stick. Gyros is meat cooked on a rotisserie and carved to order. Gyros come wrapped in pita (σε πίτα) with tzatziki, tomato, and onion. If in doubt, point and say "αυτό" — that one.
